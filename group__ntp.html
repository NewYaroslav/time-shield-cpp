Detailed Description

Facilities for retrieving time using the Network Time Protocol.

This module contains a minimal client capable of querying remote NTP servers to measure the offset between local and network time. It uses WinSock via the WsaGuard helper on Windows and POSIX sockets on Unix platforms.

Features:

  • Query NTP servers and obtain the time offset.
  • Retrieve corrected UTC time in microseconds and milliseconds.
  • Automatic WinSock initialization through WsaGuard (Windows build).
  • Requires network connectivity.

Example Usage:

time_shield::NtpClient client;
if (client.query()) {
int64_t offset = client.get_offset_us();
int64_t utc_ms = client.get_utc_time_ms();
}
